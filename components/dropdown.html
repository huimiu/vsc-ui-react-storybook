Dropdown

CSS overrides for the Fluent UI v9 <Dropdown>, <Combobox>, <Listbox>, and <Option> components. Covers three sizes, validation states, and the full option popup surface.

Import

import './overrides/vscode-dropdown-overrides.css';

Combobox

An editable dropdown with a text input. Uses .vscode-combobox on the <Combobox> root.

<Combobox className="vscode-combobox" placeholder="Search languages..."> <Option className="vscode-option">JavaScript</Option> <Option className="vscode-option">TypeScript</Option> <Option className="vscode-option">Python</Option> </Combobox>
