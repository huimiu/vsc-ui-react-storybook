Context Menu

CSS overrides for the Fluent UI v9 <Menu>, <MenuItem>, <MenuDivider>, <MenuItemCheckbox>, <MenuItemRadio>, and <MenuGroup> components. Covers icons, keyboard shortcuts, submenus, checkable items, accents, groups, and dividers.

Import

import './overrides/vscode-menu-overrides.css';

Class Reference

ClassTargetDescription
.vscode-menuMenuPopover / MenuListMenu surface container
.vscode-menu-itemMenuItemSingle menu row
.vscode-menu-item-accentModifierHyperlink-blue label for emphasis
.vscode-menu-item-indentedModifierExtra left padding for items without an icon
.vscode-menu-dividerMenuDividerHorizontal separator between groups
.vscode-menu-groupMenuGroupLogical group with automatic border between siblings
.vscode-menu-group-headerMenuGroupHeaderSection header label within a group
